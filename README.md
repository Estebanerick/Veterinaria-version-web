# Veterinaria-version-web

Para que funcione la pagina
    abrir una terminal nueva el VS Code
    ingresar a la carpeta desde la terminal "veterinaria_django"
    ingresar: python manage.py runserver
    ir al link: http://127.0.0.1:8000/

NOTA
    Solo funciona con el VS Code abierto ejecutandose python manage.py runserver
